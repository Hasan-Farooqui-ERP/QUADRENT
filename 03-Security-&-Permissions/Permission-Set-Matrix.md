# Business Central Permission Set Matrix

| Role | Permissions | Least-Privilege Notes |
|------|-------------|------------------------|
| Inventory Clerk | Read/Write Item, Bin, Warehouse Entries | No access to Posting Groups |
| Sales Processor | Create/Release Sales Orders | No access to Item Tracking Codes |
| Finance Reviewer | Read G/L, Value Entries | No posting permissions |
| Admin | Full access | Requires approval for changes |

## Access Request Workflow
1. User submits request.
2. Manager approves.
3. Admin assigns permission set.
4. Audit evidence logged.
